## 📋 Sobre o Projeto

O **Hub Osteogênese Imperfeita** é um centro de conhecimento técnico e recursos sobre Osteogênese Imperfeita (OI), desenvolvido para servir como referência para:

- 👨‍⚕️ **Profissionais de Saúde** - Médicos, enfermeiros e especialistas
- 🎓 **Pesquisadores** - Acadêmicos e grupos de estudo
- 👨‍👩‍👧 **Pacientes e Familiares** - Informações acessíveis e confiáveis
- 🏛️ **Gestores de Saúde Pública** - Dados para políticas e protocolos

## 🎯 Objetivo Principal

Organizar e disponibilizar de forma estruturada informações sobre o uso do **Ácido Zoledrônico** no tratamento da Osteogênese Imperfeita, incluindo:

- Contexto da medicação no SUS brasileiro
- Comparativos técnicos (Pamidronato vs. Ácido Zoledrônico)
- Protocolos de segurança e exames pré-dose
- Contraindicações e limitações
- Centros de referência no Brasil
- Associações internacionais e recursos científicos

## 🔧 Personalização

### Alterar Cores

Edite as variáveis CSS em `css/style.css`:

```css
:root {
    --primary: #SUA_COR_PRIMARIA;
    --secondary: #SUA_COR_SECUNDARIA;
    /* ... outras variáveis */
}
```

### Adicionar Novo Centro de Referência

Em `index.html` ou `acido-zoledronico.html`, localize a seção de centros e adicione:

```html
<div class="center-card">
    <div class="center-header">
        <i class="fas fa-map-marker-alt"></i>
        <h4>Nome do Centro</h4>
        <span class="badge badge-primary">Especialidade</span>
    </div>
    <p><strong>Local:</strong> Cidade, Estado</p>
    <p>Descrição do centro...</p>
    <a href="URL" target="_blank" class="external-link">
        <i class="fas fa-external-link-alt"></i> Site oficial
    </a>
</div>
```

### Adicionar Nova Seção ao Dossiê

1. Adicione o link no índice (`.doc-navigation`)
2. Crie a seção com `id` correspondente
3. Use a classe `.content-section` para formatação consistente
